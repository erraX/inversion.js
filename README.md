## Installation

inversion.js can be used in a browser or in a nodejs app. It can be installed via bower or npm:

```bash
> $ bower install inversionjs
```

```bash
> $ npm install inversionjs
```

## Build

Build umd
```bash
> $ npm run build:umd
```

Build umd min
```bash
> $ npm run build:umd:min
```

Build all
```bash
> $ npm run build
```

All built files will in `/dist/*`
